CI/CD Process Flow

  • 1. Developer가 Local PC에서 애플리케이션 코드를 수정한다.
  • 2. Git 명령어를 사용하여 GitHub Repository에 Push 한다.
  • 3. Jenkins가 GitHub Webhook 이벤트를 감지한다.
  • 4. Jenkins Pipeline이 자동 실행된다.
  • 5. Maven Build를 통해 WAR/JAR Artifact를 생성한다.
  • 6. Jenkins가 Ansible Playbook을 실행한다.
  • 7. Ansible이 Kubernetes Master Node에 배포 명령을 전달한다.
  • 8. Kubernetes가 Deployment와 Pod를 생성한다.
  • 9. Service를 통해 사용자 요청을 처리한다.
